# Phase 2 Letter Group Candidates (TC-20)

**Status:** Finalised per TC-23. `letter_groups.json` drafted per TC-24.

This document contains 30 candidate letter groups for the Shared Letter Mode pool. Each group contains exactly one capital, one country, and one river sharing an initial letter, and avoids the "false-pattern trap" (no two entities in a group share a real-world geographic containment).
