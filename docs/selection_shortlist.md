# Phase 1 Selection Shortlist (TC-10)

**Status:** Drafted per TC-10. Ready for TC-11/TC-12 research.

This list contains 20 UN-member states selected for the Geography Pack v0.4 MVP. 

## Regional Balance Summary
- **Europe:** 7
- **Asia:** 4
- **Americas:** 4
- **Africa:** 3
- **Oceania:** 2

## The 20 Countries

### Europe (7)
1. `AUT` - **Austria**: Core pilot entry for both DE/local variants and facts.
2. `BEL` - **Belgium**: Core pilot entry for DE/local variants (multilingual state).
3. `DEU` - **Germany**: Core pilot entry for DE/local variants.
4. `ESP` - **Spain**: Core pilot entry for DE/local variants.
5. `FRA` - **France**: Core facts pilot entry; provides a non-trivial river choice (Loire vs. Seine).
6. `ITA` - **Italy**: Core pilot entry for DE/local variants.
7. `NLD` - **Netherlands**: Core pilot entry for DE/local variants; provides a non-trivial capital choice.

### Asia (4)
8. `CHN` - **China**: Global familiarity; provides strong canonical river options.
9. `IND` - **India**: High familiarity; iconic river system (Ganges).
10. `JPN` - **Japan**: Core facts pilot entry.
11. `KOR` - **South Korea**: High familiarity; distinct initial letter representation.

### Americas (4)
12. `ARG` - **Argentina**: High regional familiarity.
13. `BRA` - **Brazil**: Anchors the Amazon river basin.
14. `MEX` - **Mexico**: High familiarity; distinct initial letter.
15. `USA` - **United States**: Global familiarity; provides a non-trivial canonical river choice.

### Africa (3)
16. `EGY` - **Egypt**: Core facts pilot entry; anchors the iconic Nile river.
17. `KEN` - **Kenya**: High regional familiarity and clear school-atlas geography.
18. `ZAF` - **South Africa**: Provides a complex multi-capital case essential for the Curation Statement.

### Oceania (2)
19. `AUS` - **Australia**: Clear canonical capital and distinct river system (Murray).
20. `NZL` - **New Zealand**: High familiarity; provides N-initial variety.
