# Phase 3 Translations Pilot Shortlist (TC-30)

**Status:** Translations drafted and verified per TC-31 and TC-32. Ready for TC-33 (merge into `entities.json`).

This document lists the 8 country IDs selected for the trilingual (EN / DE / local) pilot. The selection criterion is that the German form differs meaningfully from the English form for at least one component (country, capital, or river).

## The 8 Pilot Entries
1. **AUT (Austria):** Austria -> Österreich, Vienna -> Wien, Danube -> Donau.
2. **BEL (Belgium):** Belgium -> Belgien, Brussels -> Brüssel, Meuse -> Maas.
3. **DEU (Germany):** Germany -> Deutschland, Rhine -> Rhein.
4. **ESP (Spain):** Spain -> Spanien, Tagus -> Tajo.
5. **ITA (Italy):** Italy -> Italien, Rome -> Rom.
6. **NLD (Netherlands):** Netherlands -> Niederlande, The Hague -> Den Haag (alternative).
7. **EGY (Egypt):** Egypt -> Ägypten, Cairo -> Kairo, Nile -> Nil.
8. **ZAF (South Africa):** South Africa -> Südafrika, Cape Town -> Kapstadt (alternative), Orange -> Oranje.
