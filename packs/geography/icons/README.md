# Geography Pack Icons

**Status:** Icon design pending (Task Card TC-50, Accessibility audit TC-51).

Three SVG icons will live here:

- `capital.svg` — building or skyline silhouette.
- `country.svg` — outline or region mark.
- `river.svg` — wavy line or current stroke.

## Accessibility requirements (per Engine Blueprint §18, Pack Blueprint §3)

- Monochrome.
- Shape-distinct at 16px, 24px, and 48px.
- Legible in greyscale (colour-independent).
- Silhouettes clearly differentiable from each other.

## Sign-off

Icons must pass Task Card TC-51 (accessibility audit by an accessibility-experienced Verifier) before the pack manifest can declare `accessibility_declaration.icons_shape_distinct: true` and `icons_colour_independent: true`.
